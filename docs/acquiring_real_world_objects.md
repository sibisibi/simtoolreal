# Acquiring Real-World Objects

This document lists links for acquiring the physical objects used in the real-world DexToolBench demonstrations. Most are exact matches, while others are the closest representative links we could find.

These links are only here to help with reproducibility. We are not affiliated with the sellers or manufacturers, and product pages can change over time. The most reliable links are generally standard retailer pages such as Amazon, Home Depot, Staples, Walmart, and Target. Marketplace or less familiar links are included when they provide a useful visual match, but please check the seller, availability, dimensions, color, and return policy before purchasing.

If the exact object is unavailable, a similar object with close geometry, material, and color should still work, though you may need to collect a new mesh and pose-tracking data for the object you actually use.
